# AdeoWebTask

Challenges: to integrate third-party API, before I never had to use it, so it was new to me.

Description: it's a simple system, builded by laravel, which allows getting the weather from third-party API and assigns a special product from the database. 

Used technologies: guzzle and faker.

How to use:
1. Download or clone repository.
2. Launch composer install
3. Connect with database
4. Start application
5. In browser URL write: http://127.0.0.1:8000/api/product?city=kaunas
6. "kaunas" can be changed with any of Lithuania cities
